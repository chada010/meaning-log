# Meaning Log — 前端

基于 Vue 3 + TypeScript + Vite 构建的个人日记应用前端。

## 技术栈

- Vue 3 (Composition API)
- TypeScript
- Vite
- Pinia（状态管理）
- Vue Router
- Element Plus（UI 组件库）
- Axios

## 本地开发

```bash
npm install
npm run dev        # 启动开发服务器 http://localhost:5173
npm run build      # 生产构建
npm run type-check # TypeScript 类型检查
```

后端服务需同时启动，参见 [后端 README](../meaning-log-backend/README.md)。
